## 运维端核心功能详解

### 1. AI 模型管理
- 模型版本列表（v1.0/v1.5/v2.0）
- 准确率/mAP/精确率/召回率各项指标
- 模型上下线切换
- 训练任务触发（模拟）

### 2. 识别准确率监控
- 按垃圾分类四大家查看识别正确率趋势
- 低准确率品类告警
- 抽查最近识别记录

### 3. 消费洞察分析（AI 核心价值）
```
投放数据 → 品类映射商品 → 社区消费画像 → 商家进货建议

示例：
  虎溪花园 牛奶盒/酸奶盒 投放量高 → 乳制品消费旺盛
  → 建议便利店：多备牛奶/酸奶积分兑换品
  
  学府悦园 方便面桶/外卖餐盒 占比高 → 速食需求大  
  → 建议便利店：多备方便面/速食兑换品
```

### 4. 系统健康监控
- 设备在线率/故障设备数
- 今日识别请求量/成功率
- API 响应延迟
- 数据库连接状态
- Redis 状态

### 5. Badcase 审核
- 抽查 AI 识别错误的记录
- 人工标注正确分类
- 积累训练数据集

## AI 数据飞轮

```
居民拍照识别 ←→ AI模型自动分类
     ↓
后台汇聚投放记录（品类/时间/社区/用户）
     ↓
运维端AI分析引擎
  ├─ 品类分布 → 消费推断 → 商家进货建议
  ├─ 识别失败 → badcase标注 → 训练数据 → 模型微调
  └─ 设备异常 → 告警 → 物业处理
     ↓
模型准确率提升 + 商家精准进货 = 生态闭环
```
