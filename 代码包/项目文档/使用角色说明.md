# 智能垃圾分类监管与积分运营系统 — 使用角色说明

> 背景：重庆市沙坪坝区大学城虎溪街道 — 虎溪花园社区  
> 典型高层住宅小区，12栋楼，约3000户居民，2024年强制垃圾分类，投放点部署智能垃圾箱

---

## 角色一：物业经理 — 周经理

| 项目 | 内容 |
|------|------|
| 真实身份 | 虎溪花园物业公司经理，35岁，负责小区日常运营 |
| 登录方式 | Web 管理后台，账号 `wuye`，管理员权限 |
| 每天工作 | ① 上班第一件事打开后台看昨天投放数据 ② 发现某栋分类正确率低，安排保洁重点巡查 ③ 住户投诉垃圾箱满了，看后台满溢度派人换桶 ④ 有人恶意乱扔 → 禁用账号 |
| 所用功能 | 设备管理（在线/离线/满溢/故障监控）、用户管理（启用/禁用居民账号）、数据统计看板、月底导出 Excel 给居委会 |

## 角色二：小区居民 — 刘阿姨

| 项目 | 内容 |
|------|------|
| 真实身份 | 虎溪花园 5栋12楼退休工人，会智能手机但不精通 |
| 登录方式 | 微信小程序 |
| 每天做的事 | ① 晚饭后出门扔垃圾，扫二维码开箱 ② 扔对了听到"分类正确，+10积分" ③ 攒够300分去物业换垃圾袋 ④ 不清楚椰子壳属于什么垃圾 → 拍照查 |
| 所用功能 | 扫码投放、拍照识别分类、积分账户查看、积分商品兑换、投放记录查询 |

## 角色三：社区超市老板 — 王姐

| 项目 | 内容 |
|------|------|
| 真实身份 | 虎溪花园门口"好邻居便利店"老板娘，42岁 |
| 登录方式 | Web 商家后台，账号 `haolinju`，商家权限 |
| 每天做的事 | ① 把垃圾袋、纸巾、洗衣液上架到积分商城 ② 居民拿核销码来店里换东西，扫码核销 ③ 月底查看出货统计，哪些商品最受欢迎 |
| 所用功能 | 商品管理（上架/下架/改库存/调积分价）、订单核销、商业报告查看 |

## 角色四：街道办城管科 — 小陈

| 项目 | 内容 |
|------|------|
| 真实身份 | 虎溪街道办城管科工作人员，负责辖区内8个小区的垃圾分类考核 |
| 登录方式 | Web 管理后台，账号 `chengguan`，只读权限 |
| 每季度做的事 | ① 登录后台看各社区数据对比 ② 虎溪花园分类正确率78%没达标 → 通知物业整改 ③ 导出 Excel 报表给区城管局汇报 |
| 所用功能 | 数据统计看板、投放数据查询、导出报表 |
